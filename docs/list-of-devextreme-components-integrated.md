# List of DevExtreme Components Integrated

Here's a quick overview of the DevExtreme components we've enabled for drag-and-drop:

| Component          | Description                   |
| ------------------ | ----------------------------- |
| DxTextBox          | Standard input field          |
| DxNumberBox        | Numeric input with formatting |
| DxDateTimeControl  | Date/time picker              |
| DxCheckBox         | Boolean toggle                |
| DxSelectBox        | Dropdown selector             |
| DxDropDownComboBox | Advanced dropdown             |
| DxGrid             | Data grid display             |
| DxChart            | Visual charting component     |
| DxScheduler        | Calendar scheduling UI        |
| DxFile             | File upload                   |
| DxImage            | Static image component        |
| DxLabel            | Simple text/label display     |
| DxTimer            | Countdown/timer widget        |
| DxSwipper          | Carousel/swiper UI            |
| DxTextArea         | Multi-line input field        |
| DxDownloadLink     | File download hyperlink       |
| DxFormGrid         | Form-based grid control       |
| DxReportCatalogue  | Report catalog display        |
